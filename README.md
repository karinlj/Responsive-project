# Responsive project for Hogia 2019

### Description

Assignment for employment as frontend developer at Hogia 2019.

### Prerequisites

NodeJS

### Installation

To clone

`git clone https://github.com/karinlj/Responsive-project.git && cd Responsive-project`

To install

`npm install`

To run

`npm start`

### Built With
* React.js
* Scss
* Bootstrap 4
